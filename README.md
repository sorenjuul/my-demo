# Equipment Dashboard Demo App

## Features

- Overview of all equipment
- Filter equipment by state
- View equipment details
  - Equipment name
  - Equipment state
  - Equipment type
  - State history
- Details page for each machine
  - Machine type name
  - Machine current order
  - Order state
  - Switch machine state
  - Order queue

## Setup and run locally
```
$ npm install
$ npm run dev
```
